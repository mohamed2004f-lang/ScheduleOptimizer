مدخَلات الجدولة (JSON)

ألصق بيانات الطلبة/الجدول/التسجيلات ثم احفظ وشغّل المحسّن

الطلبة (JSON)

قائمة عناصر كل منها يحتوي student_id و student_name. مثال: [{"student_id":"1200","student_name":"أحمد"}]

أو أنشئ CSV ثم حوّله إلى JSON قبل اللصق.

الجدول الدراسي schedule (JSON)

كل عنصر يجب أن يحتوي course_name, day, time, room, instructor, semester. مثال: {"course_name":"ميكانيكا_1","day":"السبت","time":"08:00-10:00","room":"A1","instructor":"د.علي","semester":"خريف"}

التسجيلات registrations (JSON)

كل عنصر: student_id و course_name. مثال: {"student_id":"1200","course_name":"ميكانيكا_1"}

يمكنك لصق جداول من Excel بعد تحويلها إلى JSON أو استخدام أدوات تحويل خارجية.
